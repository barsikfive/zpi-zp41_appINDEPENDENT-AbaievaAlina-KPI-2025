Що таке HTML?

HTML — це мова розмітки для створення веб-сторінок.

↑ до меню ↑

Редактори коду та IDE

Редактор початкового коду — текстовий редактор для створення, та редагування початкового коду програм.

Інтегроване середовище розробки (ІСР, англ. integrated development environment або англ. IDE) — комплексне програмне рішення для розробки програмного забезпечення. Зазвичай, складається з редактора початкового коду, інструментів для автоматизації складання та відлагодження програм. Більшість сучасних середовищ розробки мають можливість автодоповнення коду.

Елементи

HTML елемент зазвичай складається із тегу початкового і тегу кінцевого, а зміст вставляється поміж ними. HTML елемент - це все від початкового тегу до кінцевого тегу.

↑ до меню ↑

Атрибути

  • Всі HTML елементи можуть мати атрибути.
  • Атрибути надають додаткову інформацію про елемент
  • Атрибути завжди вказуються в початковому тезі
  • Атрибути зазвичай входять в пари ім'я/значення, наприклад: ім'я="значення"

↑ до меню ↑

Параграфи

Параграф – це основний спосіб розділити великий текст на невеликі логічні блоки. В HTML для розмітки параграфів призначено тег <p> (від англійського слова “paragraph”). Тег <p> є блочним елементом, він завжди починається з нового рядка та займає усю ширину веб-сторінки.

Посилання

HTML посилання – це гіперпосилання.

За замовчуванням відображатиметься (у всіх браузерах):

  • невідвідане посилання підкреслено синім кольором,
  • відвідане посилання підкреслено фіолетовим кольором,
  • активне посилання підкреслено червоним кольором.

Закладки можуть бути корисними, якщо вебсторінка дуже довга. Можна помітити, наскільки ефектифно вони використані цій величезній самостійній роботі.

Зображення

Це зображення, яке використане як логотип для сайту Honey Badger.

Воно також містить карту зображення - клікабельний ніс, посилання веде до меню сторінки.

Це збільшена фавіконка цієї сторінки в браузері.

Цей блок містить фонове зображення.

Елемент <picture> містить один або кілька елементів <source> кожен з яких посилається на різні зображення через атрибут srcset. Таким чином браузер може вибрати зображення, яке найкраще відповідає поточному перегляду та/або пристрою. Кожен елемент <source> має атрибут media, який визначає, коли зображення є найбільш підходящим.
