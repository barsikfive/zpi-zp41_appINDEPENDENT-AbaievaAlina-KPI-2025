Синтаксис

Селектор вказує на елемент HTML, який ви хочете стилізувати. Блок об’яв містить одну або кілька об’яв, розділених крапкою з комою. Кожна об’ява включає назву CSS-властивості та значення, розділені двокрапкою. Об’ява CSS завжди закінчюється крапкою з комою, а блоки об’яв беруться у фігурні дужки.

Селектори

CSS селектори використовуються для "пошуку" (або вибору) HTML-елементів, які необхідно стилізувати. Можна розділити CSS селектори на п'ять категорій: прості селектори (вибір елементів на основі імені, ідентифікатору, класу), комбінаторні селектори (вибір елементів на основі визначених відношень між ними), селектори псевдокласів (вибір елементів на основі визначеного стану), селектори псевдоелементів (вибір і стилізація частини елемента), селектори атрибутів (вибір елементів на основі атрибуту або значення атрибуту).

Кольори

В CSS кольори можуть бути вказані з використанням назви, значень RGB, значень HEX, значень HSL, значень RGBA та значень HSLA:

  • tomato
  • rgb(255, 99, 71)
  • hsl(9, 100%, 64%)
  • rgba(255, 99, 71, 0.5)
  • hsla(9, 100%, 64%, 0.5)

    Контур

    Контур - це смуга, яка проводиться навколо елементів ПОЗА МЕЖАМИ, щоб зробити елемент "виокремленим".

    Контур відрізняється від меж! На відміну від межі, контур малюється поза межею елементу і може перекривати інший зміст. Крім того, контур НЕ є частиною розмірів елементу; загальна ширина та висота елемента не залежить від ширини контуру.

    Ширина

    Властивість outline-width визначає ширину контуру.

    • thin - тонкий (зазвичай 1px)
    • medium - середній (зазвичай 3px)
    • thick - товстий (зазвичай 5px)
    • визначений розмір (в px, pt, cm, em і т.д.)

    Колір

    Властивість outline-color використовується для встановлення кольору контуру.

    Скорочена властивість контуру

    Властивість outline - це скорочена властивість для встановлення наступних індивідуальних властивостей контуру: outline-width, outline-style (обов’язково), outline-color. Властивість outline вказується як одне, два або три значення зі списку вище. Порядок значень не має значення.

    Outline Offset

    Властивість outline-offset додає простір між контуром і краєм / межею елемента. Простір між елементом та його контуром прозорий.

    Текст

    CSS має багато властивостей для форматування тексту, починаючи з кольора.

    Вирівнювання

    Властивість text-align використовується для встановлення горизонтального вирівнювання тексту. Властивість text-align-last визначає спосіб вирівнювання останнього рядка тексту. Властивості direction і unicode-bidi можна використовувати для зміни напрямку тексту елемента. Властивість vertical-align встановлює вертикальне вирівнювання елемента.

    Оформлення

    Властивість text-decoration-line використовується для додавання декоративної лінії до тексту. Властивість text-decoration-color використовується для встановлення кольору декоративної лінії. Властивість text-decoration-style використовується для встановлення стилю декоративної лінії. Властивість text-decoration-thickness використовується для встановлення товщини декоративної лінії. Властивість text-decoration є скороченою властивістю. Властивість text-decoration: none; використовується для видалення підкреслення з посилань.

    Трансформація

    Властивість text-transform використовується для визначення заголовних та рядкових літер в тексті.

    Інтервал

    Властивість text-indent використовується для визначення відступу першого рядка тексту. Властивість letter-spacing використовується для визначення пробілу між символами в тексті. Властивість line-height використовується для визначення відстані між рядками. Властивість word-spacing використовується для визначення пробілу між словами в тексті. Властивість white-space визначає, як обробляються пробіли всередині елемента.

    Тінь

    Властивість text-shadow додає тінь до тексту.

    Шрифти

    В CSS є п’ять загальних сімейств шрифтів.

    • Serif шрифти - мають невеличке обведення по краях кожної літери. Вони створюють відчуття формальності та елегантності.
    • Sans-serif шрифти - мають чисті лінії (без дрібних штрихів). Вони створюють сучасний і мінімалістичний вигляд.
    • Monospace шрифти - всі літери мають однакову фіксовану ширину. Вони створюють механічний вигляд.
    • Cursive шрифти - імітують людський почерк.
    • Fantasy шрифти - це декоративні/гральні шрифти.

    Всі різні назви шрифтів належать до одного з загальних сімейств шрифтів.

    Веб-безпечні шрифти

    Веб-безпечні шрифти - це шрифти, які універсально встановлюються в усіх браузерах і на всіх пристроях. Однак 100% повністю безпечних веб-шрифтів не існує. Завжди є шанс, що шрифт може бути не знайдений або встановлений неправильно.

    Запасні шрифти

    Дуже важливо завжди використовувати резервні шрифти. Список схожих "резервних шрифтів" додається у властивості font-family. Якщо перший шрифт не працює, браузер спробує використовувати наступний, потім знову наступний і т.д. Список має закінчуватися загальною назвою сімейства шрифтів.

    Стиль

    Властивість font-style в основному використовується для визначення курсивного тексту. Властивість font-weight визначає товщину шрифту. Властивість font-variant визначає, чи має текст відображатися дрібним шрифтом. В шрифті з маленькими заголовними літерами всі рядкові літери перетворюються в великі літери. Однак перетворені великі літери відображаються меньшим розміром шрифту, ніж початкові великі літери в тексті.

    Розмір

    Властивість font-size визначає розмір тексту. Можливість керувати розміром тексту важлива в веб-дизайні. Однак не варто використовувати налаштування розміру шрифту, щоб параграфи вигляділи як заголовки, а заголовки - як параграфи. Розмір можна встановлювати як відносний, так і абсолютний.

    Google шрифти

    Можна додати посилання на спеціальну таблицю стилів в розділі head, а потім звертатися до шрифту в CSS.

    Гарна комбінація шрифтів необхідна для гарного дизайну.

    Просунутий CSS

    Заокруглення кутів

    Властивість CSS border-radius визначає радіус кутів елемента.

    Анімації

    Анімація дозволяє елементу поступово змінювати стиль з одного стилю на інший. Щоб використовувати CSS-анімацію, спочатку треба вказати кілька ключових кадрів для анімації. Ключові кадри містять стилі, які матиме елемент у певний час. З правилом @keyframes у певний час анімація поступово змінюватиметься з поточного стилю на новий. Щоб анімація працювала, треба прив'язати анімацію до елемента.

    Кнопки

    Властивість background-color змінює колір фону кнопки. Властивість font-size змінює розмір шрифту кнопки. Властивість padding змінює відступ кнопки. Властивість border-radius додає до кнопки округлені кути. Властивість border додає кольорову рамку до кнопки. Селектор :hover змінює стиль кнопки, коли на неї наведена мишу. Властивість box-shadow додає тіні до кнопки. Властивість opacity додає прозорість кнопці (створює "вимкнений" вигляд). Також можна додати властивість курсору зі значенням "not-allowed", яка відображатиме заборону, коли курсор наведено на кнопку.

    Пагінація

    Якщо веб-сайт має велику кількість сторінок, можна додати до кожної сторінки різні види пагінації.

    Колонки

    Макет із кількома стовпцями CSS дозволяє легко визначати кілька стовпців тексту, як у газетах.

    div {
    column-count: 3;
}

    Випадаючий список

    Клас може використовувати position:relative, який потрібний, щоб вміст розташовувався прямо під кнопкою спадного меню (використовуючи position:absolute). Інший клас міститиме фактичний вміст, він приховується за замовчуванням і відображатиметься при наведенні курсора.

    Ключові слова для кольорів

    Ключові слова іноді необхідні, щоб оформлення притримувалося одного стилю: transparent - прозрачний, currentcolor - потчний колір, inherit - наслідуваний колір.

    Тіні

    text-shadow задає тінь для тексту, box-shadow задає тінь для елементу.

    Переходи

    Щоб створити ефект переходу, необхідно вказати дві речі властивість CSS, до якої потрібно додати ефект, а також тривалість ефекту, і коли має відбутися зміна - перехід до нового вигляду буде плавним.

    Ефекти

    Властивість CSS text-overflow визначає, як завеликий вміст повинен відображатися користувачеві. Властивість CSS word-wrap дозволяє розбивати довгі слова та переносити їх на наступний рядок. Властивість CSS word-break визначає правила розриву рядків. Властивість CSS writing-mode визначає, горизонтально чи вертикально розміщувати рядки тексту.

    Способи підключення стилів

    Зовнішня таблиця стилів

    Зовнішні стилі визначені в елементі link всередині розділу head HTML сторінки. Файл не повинен містити жодних HTML-тегів. Файл таблиці стилів має бути збережено з розширенням .css.

    Внутрішня таблиця стилів

    Внутрішня таблиця стилів може використовуватися, якщо одна окрема HTML сторінка має свій унікальний стиль і цей стиль більше ніде не використовується на інших сторінках.

    Вбудовані стилі

    Вбудований стиль може використовуватися для застосування унікального стилю для окремого елементу на HTML сторінці. Щоб використовувати вбудовані стилі, додайте атрибут стилю до відповідного елементу (тегу). Атрибут style може містити будь-яку властивість CSS.

    Порядок використання каскадних стилів

    Вбудований стиль має найвищий пріоритет та перевизначає зовнішні та внутрішні стилі й налаштування браузера за замовчуванням.
